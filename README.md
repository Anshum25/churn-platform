# 🧠 Autonomous Customer Churn Prediction Platform

An end-to-end ML platform that ingests raw business data, 
trains multiple models, explains predictions, and recommends actions.

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, XGBoost, SHAP
- FastAPI, React
- Docker

## Project Structure
- `src/` — Core ML modules
- `api/` — FastAPI endpoints
- `dashboard/` — Frontend UI
- `notebook/` — Research & EDA
